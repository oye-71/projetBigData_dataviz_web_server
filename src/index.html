Predictor CV

Bienvenue sur le predictor CV, une application conçue par Victoria VEGAS, Gabrielle FAURIE et Etienne DUMONTET.

Le but de ce serveur est d'afficher les données des CV après leur extraction, leur export sur le cloud et leur passage dans l'algorithme de prédiction, qui nous permet de dresser des statistiques sur différents métiers, en plus de prédire le métier le plus probable pour un CV totalement anonymisé. Ce serveur permet aussi d'importer de nouvelles données prédites directement sur la base locale.

Pour importer les données, cliquez sur "obtenir les dernières prédictions". Sinon, cliquez sur "visualisation". Bonne utilisation !
